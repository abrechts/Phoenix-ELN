ELN Policies / Protocol Options

An ELN policy is an agreement among management and users of an organization on how the ELN is to be used in a consistent manner. Optimally, policies are specified by the ELN database administrator globally.  Please refer to the Policies section of ELN Admin Tool for an explanation of the individual policies.

If no server database is in use or no global policies are set, some of them also can be set locally by each individual user (via Dashboard Menu -> Settings -> Policies). However, local policies always are overridden by global ones. Therefore all Policies options of the Settings dialog are deactivated in the presence of global policies.
